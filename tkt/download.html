Installation Instructions

Windows

1. Click the Windows download link.

2. Open the downloaded EXE file and follow the instructions.

Mac

1. Click the macOS download link.

2. Open the downloaded DMG file and drag the "TenKeyType" icon to the "Applications" icon.

Linux

1. Click the Linux download link.

2. Extract the tar.gz archive somewhere convenient.

3. Open the "tkt.run" file from the terminal (e.g. with "./tkt.run"
