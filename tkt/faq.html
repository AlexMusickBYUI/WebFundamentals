What is this for?

It's for any time you need to type with one hand! If you're eating, holding a document, or anything in between, TenKeyType allows you to type one-handed while losing minimal speed.

How hard is it to use?

Because it's based on qwerty, anybody can learn to use TenKeyType in just a few minutes?

How do I use it?

The software maps certain keys and key combinations on your number pad to things that you would normally type with two hands. The first time you use it, there will be a simple tutorial for exactly which buttons to press for what.

I don't have a number pad, can I still use it?

No. A number pad is required to use the software. You could purchase an external number pad if your keyboard doesn't include one.

What software does it support?

Anywhere that you would normally type, you can use TenKeyType. Social media, office applications, messaging apps, in-game chat...if you can type in it, then you can use TenKeyType.

Can I use this for programming?

TenKeyType is designed for typing natural English, and probably won't be very comfortable to use for programming. However, you're welcome to give it a try in manual mode.

Are languages besides English supported?

Languages besides English are not officially supported. However, you can switch to manual mode to enable per-character input, allowing you to type in whatever language you want.

What is manual mode?

TenKeyType normally requires only one keystroke per letter, and it figures out what you're trying to type from context. In manual mode, pressing one key multiple times will cycle through all the letters it is assigned to. It's kind of like texting on an old cell phone.

Why can't I download the software?
