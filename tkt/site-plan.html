Overview

Purpose

TenKeyType will be a software program that will allow a user to type quickly and comfortable using only the number pad, allowing for efficient one-handed typing. This webpage will serve as a landing page providing information and eventually download links for the software.

Audience

Anybody who spends significant amounts of time typing in natural language while multitasking. In particular, there is a special appeal to those working in data entry, accounting, or other fields that require writing frequent reports based on physical documents; so mostly young to middle-aged adults working in non-technical fields.

FAQ

What is this for?

It's for any time you need to type with one hand! If you're eating, holding a document, or anything in between, TenKeyType allows you to type one-handed without losing any speed.

How hard is it to use?

Because it's based on qwerty, anybody can learn to use TenKeyType in just a few minutes.

How do I use it?

The software maps certain keys and key combinations on your number pad to things that you would normally type with two hands. The software walks you through using it during first-time setup, and you can see a general overview here.

Who can use it?

TenKeyType is available for anybody to use.

What software does it support?

Anywhere that you would normally type, you can use TenKeyType. Social media, office applications, messagins applications, in-game chat...if you can type in it, then you can use TenKeyType.

Are languages besides English supported?

Languages besides English are not officially supported. However, you can switch to manual mode to enable per-character input, allowing you to type in whatever language you want.

What is manual mode?

TenkeyType normally requires only one keystroke per letter, and it figures out what you're trying to type from context. In manual mode, pressing one key multiple times will cycle through all the letters it is assigned to. It's kind of like texting on an old cell phone.

Why can't I download the software?

If you've stumbled accross this page, you may be wondering why the download buttons all just give you a txt file. I'm creating this page as part of a school assignment, and the software itself doesn't exist yet.
